introduction

Fabriq is a light-weight, Flexbox-driven boilerplate for simple, static landing pages. With fabriq, you make use of an array of pre-defined classes in order to set-up your page elements. An example of a page built using fabriq can be viewed here.

defining a flex parent

To define an element as a flex parent, it's as simple as adding the class p, as follows:

<div id="parent" class="p">
    <!-- this is a flex parent-->
</div>

To define how the flex parent justifies and aligns its content, add classes such as jus-ctr and ali-ctr to the element, for example

<div id="parent" class="p jus-ctr ali-ctr">
    <!-- this is a flex parent with content justified and aligned to center -->
</div>

List and detail of parent classes included with fabriq:

  • p defines a flex parent with display: flex;, flex-direction: row; and flex-wrap: wrap;
  • jus-ctr will justify your contents with justify-content: center;
  • jus-end will justify your contents with justify-content: flex-end;
  • ali-ctr will align your contents with align-items: center;

defining a flex child

Atrributes such as flex-grow, flex-shrink and flex-basis are defined by classes such as g0, s0, f100. You can easily add your own classes into the CSS to suit your needs. For example:

<div id="parent" class="p">
    <div id="child" class="g0 s0 f100">
        <!-- this is a flex child with flex-grow: 0; flex-shrink: 0; flex-basis: 100; -->
    </div>
</div>

Sometimes you may want a flex child to also act as a flex parent, which can be achieved simply by adding the .p class to the child element.

List and detail of child classes included with fabriq:

  • g0 sets flex-grow: 0; on a child
  • g1 sets flex-grow: 1; on a child
  • g2 sets flex-grow: 2; on a child
  • s0 sets flex-shrink: 0; on a child
  • s1 sets flex-shrink: 1; on a child
  • f20 sets flex-basis: 20%; on a child
  • f40 sets flex-basis: 40%; on a child
  • f60 sets flex-basis: 60%; on a child
  • f80 sets flex-basis: 80%; on a child
  • f100 sets flex-basis: 100%; on a child
  • f33 sets flex-basis: 33.33%; on a child
  • f66 sets flex-basis: 66.66%; on a child

buttons with fabriq

Buttons come in two styles with fabriq. An alternative style can be defined using .button-alt. All types of buttons are included in the definitions.

For example:

<div class="p jus-ctr ali-ctr">
    <input type="button" value="Button Default">
    <input class="button-alt" type="button" value="Button Alt">
</div>

forms with fabriq

For ease, forms come pre-styled with fabriq, all you need to do is change how they look!

The following code produces the example below:

<form class="p jus-ctr ali-ctr">
    <div class="g0 s0 f80">
        <label for="email">e-mail</label>
        <br>
        <input type="text" name="email" placeholder="your e-mail">
        <br>
        <label for="query">query</label>
        <br>
        <select>
            <option selected disabled hidden>your choice</option>
            <option value="option-1">vanilla</option>
            <option value="option-2">chocolate</option>
            <option value="option-3">strawberry</option>
        </select>
        <br>
        <label for="message">message</label>
        <br>
        <textarea placeholder="your message"></textarea>
        <br>
        <div class="p jus-ctr ali-ctr">
            <input type="submit" value="submit">
            <input class="button-alt" type="reset" value="reset">
        </div>
    </div>
</form>
